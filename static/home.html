The Office Quotes

API endpoint features:

  • All quotes
  • Quotes by ID
  • Quotes by person
  • One random quote
  • N random quotes

The endpoint, https://office-quote-api.herokuapp.com/quotes returns alls quotes in the table. Try Me!

The endpoint, https://office-quote-api.herokuapp.com/quotes/id/25 returns the quote with the specified id. Try Me! 


[
    {
        "id": 25,
        "person": "Michael",
        "quote": "Come on, guys. Early worm gets the worm."
    }
]

The endpoint, https://office-quote-api.herokuapp.com/quotes/person/Jim returns all quotes by Jim. Try Me!

The endpoint, https://office-quote-api.herokuapp.com/quotes/random returns one random quote. Try Me! 


[
    {
        "id": 66,
        "person": "Dwight",
        "quote": "Who is Justice Beaver?"
    }
]

The endpoint, https://office-quote-api.herokuapp.com/quotes/random/3 returns 3 random quotes. Try Me!

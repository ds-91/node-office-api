The Office Quotes

API endpoint features:

  • All quotes
  • Quotes by ID
  • Quotes by person
  • One random quote
  • N random quotes

Calling the api on https://office-quote-api.herokuapp.com/quotes returns alls quotes in the table.

Calling the api on https://office-quote-api.herokuapp.com/quotes/id/1 gives you: 


{
    "id": 1,
    "person": "Michael",
    "quote": "Would I rather be feared or loved? Easy. Both. I want people to be afraid of how much they love me."
}

Calling the api on https://office-quote-api.herokuapp.com/quotes/person/Michael gives you all quotes by Michael.

Calling the api on https://office-quote-api.herokuapp.com/quotes/random gives you one random quote.

Calling the api on https://office-quote-api.herokuapp.com/quotes/random/3 returns 3 random quotes.
